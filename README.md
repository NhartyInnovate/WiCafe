A Flask web application that lists laptop-friendly cafes with WiFi, sockets, and good coffee.  
Users can browse cafes, search by location, and view amenities before deciding where to work or study.

---

## 🔗 Live Demo

[![Live Demo](https://img.shields.io/badge/Live-Demo-green)](https://wi-cafe.vercel.app/)

> Note: The deployed version is **read-only** due to SQLite limitations on serverless platforms. Add/Edit/Delete functionality works when running locally.

---

# Features

### ☕ Browse Cafes
View cafes with useful information such as:

- Location
- Seating availability
- WiFi availability
- Power sockets
- Toilet availability
- Call-friendly environment
- Coffee price
- Map link

---

### 🔍 Search Cafes

Search cafes by:

- Cafe name
- Location

---

### ➕ Add New Cafes

Users can submit new cafes including:

- Cafe name
- Image URL
- Location
- Amenities
- Coffee price
- Google Maps link

---

### 🔐 Admin Editing

Admin users can:

- Edit cafe details
- Delete cafes

Admin login reveals edit controls on the interface.

---

### 🎨 Modern UI

The interface uses:

- Bootstrap
- Custom CSS
- Responsive design
- Card-based layout for cafes

---
